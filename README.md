# element

[![dependency status][3]][4]

[![browser support][5]][6]

Turn HTML into DOM elements x-browser.

```js
var element = require("element")

element("<p>Hello</p><p>World</p>")[0]
// => Hello

element("<p>Hello</p><p>World</p>")[1]
// => World

```

## Installation

`npm install element`
